# BSRU Spider Specimen Catalog Website

This is a static Github site for browsing and searching a historical museum spider specimens cataloguing under the Bee and Spider Research Unit (BSRU), Chulalongkorn University, Thailand. The catalog was retrieved from a CSV database.

## Overview

The website is intended for museum, natural history, biodiversity, and collection-based datasets, especially historical spider specimen records. 
The website also served as a fullfillment for a bachelor senior project for Department of Biology, Faculty of Science, Chulalongkorn University in 2026.


## Main Features

- Static website
- Loads specimen records directly from a CSV file
- Clean header, sidebar, and main table layout
- Search and filter tools in the left panel
- Multi-word search support
